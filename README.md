## 📌 Project Overview

This project is an **AI-based Quiz Application** built using:

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL (via XAMPP)

It allows users to take quizzes while the system intelligently selects and verifies questions using AI models.

⚙️ Features

* ✅ AI-generated quiz questions
* ✅ Multiple model verification system
* ✅ Dynamic question generation, AI based question generat
* ✅ Clean UI (HTML/CSS/JS)
* ✅ API-based architecture
* ✅ Database integration (MySQL)

## 🤖 AI Working Flow

1. Load available AI models
2. Verify working models
3. Generate quiz questions
4. Validate answers using multiple models
5. Display results to user
